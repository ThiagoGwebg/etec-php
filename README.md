## ⚙️ How to Run the Project

To run the PHP files locally you need a local development server.

### 1️⃣ Clone the repository

git clone https://github.com/ThiagoGwebg/etec-php.git

---

### 2️⃣ Open the project folder

cd etec-php

---

### 3️⃣ Run with a local server

You can use tools such as:

* XAMPP
* WAMP
* Laragon
* PHP built-in server

Example using the PHP built-in server:

php -S localhost:8000

Then open in your browser:

http://localhost:8000
